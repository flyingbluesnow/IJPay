# 安装依赖

# Maven

一次性添加所有支付方式的依赖

在项目的 pom.xml 的 dependencies 中加入以下内容:

<dependency>
    <groupId>com.github.javen205</groupId>
    <artifactId>IJPay-All</artifactId>
    <version>latest-version</version>
</dependency>
