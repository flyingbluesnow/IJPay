# 微信系开发脚手架

# 简介

TNWX: TypeScript + Node.js + WeiXin 微信系开发脚手架，支持微信公众号、微信支付、微信小游戏、微信小程序、企业号/企业微信、企业微信开放平台。最最最重要的是能快速的集成至任何 Node.js 框架(Express、Nest、Egg、Koa 等)

# 简易指南

# 尝鲜版

$ git clone https://gitee.com/javen205/TNWX.git
$ cd TNWX
$ yarn bootstrap
$ yarn tsc
